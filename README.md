# Glaucoma Detection AI

A web-based deep learning application for automated glaucoma screening using retinal fundus images.

## Overview

Glaucoma is one of the leading causes of irreversible blindness worldwide. Early detection is essential for timely treatment and vision preservation. This application leverages a lightweight deep learning model to analyze retinal fundus images and predict the likelihood of glaucoma.

The project demonstrates the complete AI development lifecycle, from model training and evaluation to deployment as an interactive web application.

## Features

* Upload retinal fundus images through a simple web interface
* Automated glaucoma prediction
* Real-time inference
* Lightweight and efficient deep learning model
* Publicly accessible via Hugging Face Spaces

## How It Works

1. Upload a retinal fundus image.
2. The image is automatically preprocessed.
3. The trained deep learning model analyzes the image.
4. The application returns:

   * Diagnosis (Glaucoma / No Glaucoma)
   * Prediction confidence score

## Model Information

The deployed model is based on a lightweight CNN architecture trained for binary glaucoma classification from retinal fundus photographs.

The system was designed as a screening-oriented solution to support the early identification of glaucoma cases while maintaining computational efficiency.
